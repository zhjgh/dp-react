## API文档
| 属性            | 说明           | 类型                        | 默认值        | 是否必传 |
|---------------|--------------|---------------------------|------------|------|
| action        | 上传图片请求地址     | string                    | -          | 是    |
| headers       | 请求头参数        | {}                        | -          |      |
| data          | 请求参数         | {}                        | -          |      |
| maxSize       | 文件大小限制       | number                    | 2（单位M）     |      |
| isShowImg     | 是否显示图片       | boolean                   | false      |      |
| initialValues | 初始化数据        | []                        |            |      |
| isCanRemove   | 是否可以删除       | boolean                   | true       |      |
| multiple      | 是否可以同时上传多个图片 | boolean                   | false      |      |
| isDragSort    | 是否开启拖拽排序     | boolean                   | false      |      |
| maxCount      | 限制图片张数       | number                    | 1          |      |
| direction     | 图片展示方向       | 'horizontal' | 'vertical' | horizontal |      |
