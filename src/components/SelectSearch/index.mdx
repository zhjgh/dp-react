## API文档
| 属性              | 说明                 | 类型       | 默认值            | 是否必传 |
|-----------------|--------------------|----------|----------------|------|
| action          | 请求地址               | string   | -              | 是    |
| headers         | 请求头参数              | {}       | -              |      |
| fetchData       | 请求附加参数             | {}       | -              |      |
| pageSize        | 分页数量               | number   | 10             |      |
| debounceTimeout | 延迟请求时间（单位毫秒）       | number   | 800            |      |
| labelAndValue   | 指定label和value的参数名称 | string[] | ['name', 'id'] |      |

## 接口返回数据格式
```
{
  state: 1, // 请求成功
  list: [], // 表格数据
  page: {
    current: 1, // 第几页
    pageSize: 10, // 每页数量
    total: 200,  // 数据总量
    totalPage：20 // 分页总量（必须返回）
  }
}
```
