# SearchGroup 搜索
#### 默认

<Playground>
  {() => {
    const [searchParams, setSearchParams] = React.useState()

    const fields = [
      { name: '姓名', label: 'name' },
      {
        name: '性别',
        label: 'type',
        type: 'select',
        options: [
          { label: '男', value: '0' },
          { label: '女', value: '1' },
        ],
      },
      { name: '日期', label: 'date', type: 'datePicker' },
      {
        name: '日期范围',
        label: ['dateStart', 'dateEnd'],
        type: 'rangePicker',
      },
    ]

    const onSearch = (values) => {
      console.log(values)
      setSearchParams(values)
    }

    return (
      <SearchGroup fields={fields} onSearch={(values) => onSearch(values)} />
    )
  }}
</Playground>

## API文档
| 属性          | 说明                    | 类型     | 默认值 | 是否必传 |
|-------------|-----------------------|--------|-----|------|
| fields      | 表单项配置                 | []     | -   | 是    |
| onSearch    | 查询方法                  | func   | -   | 是    |
| fetchParams | 请求附加参数                | {}     | -   |      |
| maxNum      | 表单项显示最大数量，超过隐藏，显示更多按钮 | number | 8   |      |
